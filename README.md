# 📦 DTDC Invoice Checker

A Python-based web application to verify and audit DTDC courier invoices based on predefined agreement rules.

---

## 🚀 Project Overview

The **DTDC Invoice Checker** helps in validating whether the invoice generated by DTDC matches the agreed pricing and rules.

It compares:
- Charged amount in invoice
- Expected amount based on agreement

and identifies:
- ✅ Correct charges  
- ⚠️ Overcharged invoices  
- ⚠️ Undercharged invoices  

---

## 🛠️ Features

- 📄 Upload invoice (PDF / CSV)
- 🔍 Extract invoice data automatically
- 📊 Calculate expected charges using agreement rules
- ⚖️ Compare invoice vs calculated values
- 🚨 Highlight mismatches
- 📑 Generate report (optional)

---

## 📦 Tech Stack

- Python
- Flask
- Pandas
- PDFPlumber (for PDF extraction)
- PyTesseract (for OCR)
- pdf2image (for image-based PDFs)

## 📊 Calculation Logic (DTDC)

The system calculates charges based on:

* 📦 Weight (Actual / Volumetric)
* 🌍 Zone mapping (Origin → Destination)
* 💰 Rate per KG (Zone-wise)
* 🔥 Fuel surcharge (FSC)
* 📄 Docket charges
* ➕ Additional charges (if applicable)

Final amount is compared with invoice total.

## ⚠️ Common Issues

* ❌ `pdfplumber not found`
  → Run: `pip install pdfplumber`

* ❌ OCR not working
  → Install Tesseract and set path:

  ```
  C:\Program Files\Tesseract-OCR\tesseract.exe
  ```
